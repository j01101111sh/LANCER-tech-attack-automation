# Lancer Tech Attack Automation

A Foundry VTT module designed to streamline and automate standard Tech Attacks and electronic warfare actions for the [Lancer TTRPG system](https://github.com/Eranziel/foundryvtt-lancer).

⚠️ **Compatibility Notice:** This module is designed exclusively for **Foundry VTT Version 12**, in line with the current compatibility of the Lancer game system.

## Features

This module automates several common Invade options and tech actions, reducing the manual book-keeping for GMs and players. Based on the included scripts, it currently supports automation for:

- **Fragment Signal:** Adds a description for this invade option with drag-and-droppable statuses.
- **Eject Power Cores:** Turns the Jammed conditions in the description to drag-and-droppable statuses to easily apply them on a hit.
- **Data Siphon:** Automates the effects of the Data Siphon trait of the Chomolungma by adding a button to scan after every tech attack for attackers with the trait.
- **Balance Control Lockout:** Turns the Prone and Immobilized conditions in the description to drag-and-droppable statuses to easily apply them on a hit.

## Usage

Once the module is active, the automated scripts hook directly into the Lancer system's attack resolution flow.
